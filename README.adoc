=== Remove passwd
```
sudo passwd -d root
sudo passwd -d smalinux
```

=== Main setup
```
sudo apt update
yes | sudo apt install git

cd ~
git clone https://github.com/smalinux/dotfiles

cd dotfiles
./dotfiles.sh

```

### Install coc.vim
```
sudo -s
curl -sL install-node.now.sh/lts | bash
sh ~/dotfiles/coc.vim.sh
```
